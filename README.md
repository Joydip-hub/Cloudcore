# Normal Calculator

A smooth, installable calculator app built with normal HTML, CSS, and JavaScript structure.

## Features

- Addition, subtraction, multiplication, division, and modulo.
- C/C++ style operator functions and expression evaluation.
- Operator precedence for accurate calculations.
- Decimal, negative number, keyboard, clear, and backspace support.
- Installable Progressive Web App with offline caching.

## Run locally

Open `index.html` directly, or serve the folder with any static server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in a browser.

## Download/install the app

When served through a browser that supports Progressive Web Apps, use the `Download App` button or the browser install option to install the calculator on your device.
